 .lumen (люмены) - int или double, в зависимости от того, требуется ли вам дробное значение.
  .resolution (разрешение) - обычно представляется в виде строки (String),
             так как разрешение часто записывается как “ширина x высота” (например, “1920x1080”).
  .contrastRatio (контрастность) - также обычно представляется в виде строки (String),
             так как контрастность часто записывается как “1:x” (например, “1:1000”).
  .lampLife (срок службы лампы) - int, если срок службы измеряется в часах.
  .projectionSize (размер проекции) - double, если размер проекции измеряется в дюймах или сантиметрах.
  .connectivity (подключение) - String, так как тип подключения обычно описывается словами
             (например, “HDMI”, “USB”, “Wi-Fi”).
  .is3DReady (поддержка 3D) - boolean, так как это бинарное значение (да или нет).
  .scanResolution (разрешение сканирования): обычно измеряется в точках на дюйм (dpi),
              поэтому подходит тип данных int.
  .scanSpeed (скорость сканирования): может быть представлена как количество страниц в минуту,
             так что int будет подходящим выбором.
  .colorDepth (глубина цвета): измеряется в битах на цвет и обычно является целым числом, так что int подходит.
  .maxPaperSize (максимальный размер бумаги): часто представляется в виде строкового значения,
             например “A4”, “A3” и т.д., поэтому String.
  .isDuplexScanning (двустороннее сканирование): булево значение,
         указывающее на поддержку или отсутствие поддержки двустороннего сканирования, поэтому boolean.
  .scannerType (тип сканера): может описывать тип сканера, например, планшетный, барабанный и т.д., поэтому String.

  .printTechnology (технология печати): String - обычно описывается словами, например, “лазерная” или “струйная”.
  .printSpeed (скорость печати): int или double - если скорость измеряется в страницах в минуту и может быть дробной.
  .maxResolution (максимальное разрешение): String - разрешение часто указывается
                                  в виде “ширина x высота”, например “2400x1200”.
  .isColor (цветной): boolean - указывает, поддерживает ли принтер цветную печать.
  .paperSize (размер бумаги): String - размеры бумаги обычно обозначаются стандартными названиями,
                                  такими как “A4” или “Letter”.
  .connectivity (подключение): String - типы подключений, такие как “USB”, “Ethernet”, “Wi-Fi”.
  .dutyCycle (рабочий цикл): int - обычно указывается максимальное количество страниц,
                                которое можно напечатать в месяц.

  .screenSize (размер экрана): double - если размер экрана измеряется в дюймах и может быть дробным числом.

  .panelType (тип панели): String - тип панели описывается словами, например “IPS” или “TN”.
  .refreshRate (частота обновления): int - частота обновления измеряется в герцах и обычно является целым числом.
  .isCurved (изогнутый): boolean - указывает, изогнут ли монитор или нет.
  .ports (порты): String[] - массив строк, так как монитор может иметь несколько портов,
                            например [“HDMI”, “DisplayPort”].
  .isHDR (поддержка HDR): boolean - указывает, поддерживает ли монитор HDR.

  .processorType (тип процессора): String - обычно описывается словами, например, “Intel Core i7” или “AMD Ryzen 5”.
  .ramSize (размер оперативной памяти): int - если размер оперативной памяти измеряется
                            в гигабайтах и является целым числом.
  .storageSize (размер хранилища): int или double - если размер хранилища измеряется
                            в гигабайтах или терабайтах и может быть дробным числом.
  .batteryLife (время работы от батареи): int или double - если время работы от батареи
                            измеряется в часах и может быть дробным числом.
  .screenSize (размер экрана): double - если размер экрана измеряется в дюймах и может быть дробным числом.
  .operatingSystem (операционная система): String - обычно описывается словами,
                            например, “Windows 10” или “macOS Catalina”.
  .isTouchscreen (сенсорный экран): boolean - указывает, имеет ли ноутбук сенсорный экран.

  .id: String - идентификатор обычно представляет собой уникальный идентификатор,
                           который может быть комбинацией букв и цифр.
  .modelName (модель): String - название модели обычно представляет собой комбинацию букв и цифр.
  .manufacturer (производитель): String - название производителя обычно представляет собой строку.
  .yearOfManufacture (год выпуска): int - год выпуска обычно представляет собой четырехзначное целое число.
  .price: double - цена может быть дробным числом.
  .weight: double - вес обычно измеряется в килограммах и может быть дробным числом.
  .isOperational (исправность): boolean - указывает, в рабочем состоянии ли устройство.

  .isCellular (сотовая связь): boolean - указывает, поддерживает ли устройство сотовую связь.
  .cameraResolution (разрешение камеры): String - разрешение камеры обычно представляется
                               в виде комбинации чисел, например “12 MP”.

  .speed (скорость): int или double - если скорость измеряется в Мбит/с и
                               может быть дробным числом.
  .band (диапазон): String - обычно описывается словами, например, “2.4 ГГц” или “5 ГГц”.
  .security (безопасность): String - типы безопасности, такие как “WPA2” или “WPA3”.
  .ports (порты): int - количество портов на устройстве.
  .range (диапазон): int или double - диапазон сети, обычно измеряется в метрах или футах.
  .isDualBand (двухдиапазонный): boolean - указывает, поддерживает ли
                                маршрутизатор работу в двух диапазонах (2,4 ГГц и 5 ГГц).
  .connectivityType (тип подключения): String - тип подключения, например, “Ethernet” или “Wi-Fi”.

  .portCount (количество портов): int - количество портов на устройстве.
  .layer (слой): int - слой коммутатора в модели OSI, обычно 2 (для коммутаторов)
                                или 3 (для маршрутизаторов).
  .managementType (тип управления): String - тип управления, например,
                               “управляемый” или “неуправляемый”.
  .powerOverEthernet (Power over Ethernet): boolean - указывает, поддерживает ли коммутатор PoE.
  .stackable (стыкуемый): boolean - указывает, можно ли объединять несколько таких устройств
                                 в стек.
  .formFactor (форм-фактор): String - форм-фактор устройства, например,
                                “настольный” или “для монтажа в стойку”.
